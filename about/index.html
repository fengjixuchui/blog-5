About Me

About Me!

My name (or “persona”) is Hifumi1337, I am a Cybersecurity Engineer who also enjoys working in Game Development and Software Engineering. I’ve spent over 7 years in total learning and educating myself on various topics and touching multiple areas of technology. I am definitely a red teamer by heart, Penetration Testing is always my go-to in any assessment, but I am also an advent researcher. I don’t always have as much time as I would like, but I love researching vulnerabilities, malware analysis, and pretty much just picking things apart to see how they work.

As I mentioned before, I am also a Game Developer and Software Engineer. I’ve worked many roles in my life, everything from Information Technology roles to Web Developement, and those are just a few. At the moment, Game Development is mainly just a hobby, but it’s definitely something I would consider professionally, if I’m passionate enough about the project.

Thanks for sticking around! Please feel free to explore the blog, click on posts, and all that jazz! :)

Projects

Below you will find some of the projects I’ve completed that I feel are pretty great! I have many more projects that are not listed here.

Universal CTF

Universal CTF is a project I started to help improve CTF experiences for beginners and speed up gathering results for veterans. Thanks to Universal CTF, you will no longer have to type out long commands over and over again, now you have a beautiful terminal based program to handle it all for you!

Encrypt It

Encrypt-It is a simple password generator program that runs in the command line, offerring mulitple encoding and encryption methods.

LSASS Dumper

Simple LSASS Dumper created using C++ as an alternative to using Mimikatz memory dumper. Running this program dumps data from the LSASS process on Windows and then Mimikatz parses the data.

Rockbell

Multithreaded HTTP server built in Rust.
Rockbell is a barebones HTTP (GET) server that renders HTML, CSS, and JS files.

NES Emulator (WIP)

NES emulator built in Rust. Currently, this emulator can only run the Snake game (machine code). I am working on adding more functionality as time permits!
